# Theming the Upup Uploader

The uploader is themed three ways, all via the `theme` prop:

- **Mode** — `theme={{ mode: 'light' | 'dark' | 'system' }}`. Use the toolbar
  paint-roller to preview light/dark.
- **Tokens** — `theme={{ tokens: { color: { primary: '#1849d6' } } }}`.
- **Slots** — per-element class overrides, e.g.
  `theme={{ slots: { 'fileList.uploadButton': 'bg-indigo-600 text-white' } }}`.

Open **React/Appearance** for live, control-driven examples.
