Playground UX mockups

Three concrete proposals for making the playground more discoverable. Click a card to see the mock. These are pure HTML — not wired to the live uploader.

#1
Inline prop docs
An (i) icon beside each prop opens a popover with the description and a link to the docs page. Self-documenting without leaving the playground.
Highest ROI · smallest change #2
Live event log
A panel below the preview that streams every event the uploader fires, with timestamps and payloads. Shows developers what actually happens during an upload.
Reveals behavior #3
Preset quick-starts
A row of opinionated starting-point buttons (S3 minimal, Managed mode, Multi-cloud, Mini, UI slots, Processing). One click seeds a realistic config.
