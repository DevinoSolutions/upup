# Theming the Upup Uploader (Vue)

Identical theming contract to React — pass the `theme` prop:

- **Mode** — `:theme="{ mode: 'dark' }"`
- **Tokens** — `:theme="{ tokens: { color: { primary: '#1849d6' } } }"`
- **Slots** — `:theme="{ slots: { 'fileList.uploadButton': 'bg-indigo-600 text-white' } }"`

Open **Vue/Appearance** for live, control-driven examples.
